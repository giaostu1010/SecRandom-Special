v3.0 - Nonomi (十六夜野宫) alpha 1

## 🚀 主要更新

- 新增 **安全设置页面**，实现基础安全验证功能
- 新增 **安全设置导航**，支持侧栏切换
- 新增 **抽取设置页面**，实现点名/闪抽/抽奖设置功能
- 新增 **抽取设置导航**，支持侧栏切换
- 新增 **人脸抽取设置**，支持配置子页
- 新增 **联动设置页面**，对齐旧版功能

## 💡 功能优化

- 优化 **默认语言**，默认简体中文
- 优化 **字体设置**，支持系统字体
- 优化 **字体设置**，支持输入搜索
- 优化 **设置配置**，按分类存储

## 🐛 修复问题

- 修复 **设置页多语言**，补齐资源初始化
- 修复 **语言切换**，支持切回英文
- 修复 **语言切换**，运行时切换不生效
- 修复 **字体配置**，兼容旧版索引
- 修复 **配置文件**，移除冗余字段
- 修复 **抽取设置保存**，支持子项持久化
- 修复 **抽取设置导航**，补齐子页面

## 🔧 其它变更

- 无
